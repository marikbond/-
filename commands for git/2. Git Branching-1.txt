
1 Просмотр имеющихся веток
	#git branch  			выводит список доступных для переключения веток
	#git branch -v  		дает посмотреть последний коммит на каждой из веток
	#git branch --merged	показывает те ветки которые уже были слиты с текущей
	#git branch --no-merged показывает все ветки, содержащие наработки, которые еще не объединили с текущей веткой
	
2 Создание новой ветки:
	
	#git branch new-branch	создает ветку с именем new-branch
	
	#git checkout -b new-branch	создает ветку с именем new-branch и перебрасывает на нее HEAD (переключается)
	
3. Удаление веток. Небходимо переключится на другую ветку и выполниь команды:
	
	#git branch -D / -d new-branch 	удалит ветку с названием new-branch
		-d проверяет находимся ли мы на той ветке которую удаляем
		-D удаляет ветку без разбора. 
		
4. Слияние. Для слияния необходимо перейти на ту ветку, в которую необходимо выполнить 
	слияние и выполнить команду git merge на которую необходимо вливать изменения и 
	выполнить команду git merge new-branch.
